\section{Saving log messages to files}
\textit{mclogd} saves log messages received via the loopback
and multicast to local files.  Filters may be used to select
which messages are saved to a given log file path.  Log file
paths may use message contents in their construction:
\texttt{\%H} will be replaced with the source hostname,
\texttt{\%I} will be replaced with the application identifier
(program name), and \texttt{\%F} will be replaced with the
log facility (\texttt{daemon}, \texttt{user}, \texttt{local0},
et. al.).
