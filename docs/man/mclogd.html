NAME

mclogd — logging daemon

SYNOPSIS

mclogd [-c configFile] [-d] [-p pidFile]

DESCRIPTION

mclogd saves log messages received via the loopback interface and/or encrypted multicast to files, and sends log messages via encrypted multicast.

The following command line options are available:

-c configFile
Specify the mclogd(8) configuration file. If not specified, /usr/local/etc/mclogd.cfg is used.
-d
Do not daemonize. Useful for debugging.
-p pidFile
Specify the path to the file where mclogd(8) should store its process ID.

STARTING AT BOOT

FreeBSD

/usr/local/etc/rc.d/mclogd is installed as part of the FreeBSD package. It is a standard FreeBSD start/stop script. To enable mclogd(8) at boot, set mclogd_enable to "YES" in /etc/rc.conf:

mclogd_enable="YES"

Linux (with systemd)

/etc/systemd/system/mclogd.service is installed as part of the Linux package. It is a standard systemd service configuration file. To enable mclogd(8) at boot:

systemctl enable mclogd

To start it once enabled without rebooting:

systemctl start mclogd

macOS

/Library/LaunchDaemons/net.mcplex.mclogd.plist is installed as part of the macOS package. It should be started during installation, but if not, it can be started with:

launchctl load -w /Library/LaunchDaemons/net.mcplex.mclogd.plist

FILES

CONFIGURATION FILE

The standard location of the configuration file is /usr/local/etc/mclogd.cfg

An example configuration is included in /usr/local/etc/mclogd.cfg.example

mclogd will reload the configuration on the reception of SIGHUP.

CREDENCE KEY FILES

mclogd uses libCredence for encryption and authentication. Hence it requires credence key files. The location of these key files is specified as the keyDirectory in the mclogd configuration file. See the credence(1) manpage for information on creating the public and private key files.

<keyDirectory>/id_ed25519
The mclogd(8) private key file, created with credence(1). This file should be owned by root and have permissions 0600. It must contain the private part of an Ed25519 RSA key pair.
<keyDirectory>/id_ed25519.pub
mclogd(8's) public key file, created with credence(1). This file should be owned by root and have permissions 0600. It must contain the public part of an Ed25519 key pair.
<keyDirectory>/known_keys
This file must contain the public keys of users allowed to query mclogd(8) with mclog(1) (one per line).
